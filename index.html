Statistical Modelling · R

Predicting Trophy Striped Bass Conditions

Built a logistic regression model in R using 8 environmental variables (water temperature, weather patterns, seasonal trends) by analyzing 5 years of striped bass catch data. Achieved 0.786 AUC with 70.7% test accuracy via systematic variable selection and train/test validation.

API Integration · Python

New England Ski Resort Daily Ranker (SkiRank)

A Python CLI tool that pulls live forecast data from 39 New England ski resorts and ranks current conditions using a weighted scoring algorithm. Integrates the National Weather Service API across CT, MA, NH, ME, and VT, scoring each resort on a 100-point scale based on wind, precipitation, and temperature.

Experimental Design · R

The Effect of Jumpscares on Word Search Performance

Designed and executed a randomized controlled experiment testing 30 subjects to investigate the impact of auditory jumpscares on cognitive task performance. Subjects were randomly assigned to three treatment conditions (JA, JNA, NJ) with covariates for sex and class year. Used ANOVA testing to determine statistical significance.
